Namespace RueI

The RueI namespace contains classes shared across all of RueI. Generally, you shouldn't need to use the classes and interfaces within this namespace. The exception is the RueIMain class that contains the EnsureInit() method, which should be called by any plugin using RueI upon being loaded.
