Remarks

An AutoElement puts an Element in a DisplayCore if they match a Roles. You can use the AutoElement(Roles, Element) constructor for an AutoElement that assigns a single instance of an Element. On the other hand, if you want to create an element for each player, you can use the AutoElement(Roles, Func<DisplayCore, Element>) constructor.
